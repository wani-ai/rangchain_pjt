```
- 콘다 가상환경 만들고, 활성화 하기
conda create -n lc_env python=3.12
conda accitvate lc_env

-작업디렉토리 이동
cd lnagchaon_pjt
```

- 마크다운
pip install tabulate

- 웹 backend, frontend 개발
pip install streamlit

streamlit run main_app.py
